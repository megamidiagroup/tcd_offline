Parametros

idCategory = valor para o id da serie ou categoria, exemplo definido: idCategory:13, default = 22

idContent = valor para o id do programa, exemplo definido: idContent:581, default = 512

playAuto = true ou false para iniciar player execução, default = false

repeat = reprodução aleatórias dos audios, default = false

base_url = local onde buscar a api do projeto, exemplo definido: base_url:http://audio.sebrae.com.br/, default = http://audio.sebrae.com.br/
